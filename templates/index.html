This is demo to search in RDF data file. I have taken one simple RDF from FigShare.com . The data is a RDF file. I have used the "rdflib" package of python for accessing the RDF file. It is a very basic demo to practice interaction with a rdf using python library and get the result on the web page. Flask, a python framework for creating web apps, is used to provide an interface to search person's name and in return Full name and their corresponding email address are returned. SPARQL select query is used to fetch the data. The dataset is small and only contains the names and the email addresses of the people. The demo shows the interaction between RDF and web page interface using "rdflib"and "SPARQL" using Flask.
